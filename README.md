# film-site-information-archive
该仓库主要用于建立影视类网站的信息档案，对公开网络中存在的电影与电视剧相关站点进行基础资料整理。记录内容涵盖站点名称、链接地址、站点类别及简要说明，旨在形成可长期维护的影视站点信息库。本项目仅作信息汇总与资料存档使用，不涉及任何影视内容的存储或传播。所有信息均来自公开渠道，不保证实时准确性，仅供学习与研究参考使用，与所记录网站不存在任何隶属或合作关系。
